# Chess React

Шахматная игра, реализованная на React и TypeScript.

## Описание

Интерактивное шахматное приложение с полным набором фигур, валидацией ходов, таймером для каждого игрока и отображением сбитых фигур.

## Возможности

- ✅ Полная шахматная доска 8x8
- ✅ Все шахматные фигуры (пешки, ладьи, кони, слоны, ферзи, короли)
- ✅ Валидация ходов для каждой фигуры
- ✅ Подсветка доступных ходов
- ✅ Поочередная игра (белые/черные)
- ✅ Таймер для каждого игрока (5 минут по умолчанию)
- ✅ Отображение сбитых фигур
- ✅ SVG иконки для всех фигур

## Технологии

- **React** 19.1.1
- **TypeScript** 4.9.5
- **React Scripts** 5.0.1

## Установка

```bash
npm install
```

## Запуск

### Режим разработки

```bash
npm start
```

Приложение откроется по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена

```bash
npm run build
```

### Запуск тестов

```bash
npm test
```

## Как играть

1. Выберите фигуру своего цвета (белые ходят первыми)
2. Доступные ходы будут подсвечены
3. Кликните на подсвеченную ячейку, чтобы сделать ход
4. Игра автоматически переключается между игроками
5. Используйте кнопку "Restart game" для перезапуска партии

## Особенности реализации

- Объектно-ориентированный подход с классами для фигур
- Каждая фигура имеет свой метод `canMove()` для валидации ходов
- Доска обновляется через копирование состояния
- Подсветка доступных ходов при выборе фигуры
